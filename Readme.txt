Enzo - Portale automobilisto crowdsourcing
-------------------------------------------------------
Istruzioni per il setup

Una volta scompattato l'archivio copiare la cartella "enzo"
dove si ritiene più opportuno, al suo interno vi è l'intero
portale.

Per provarlo in locale è possibile entrare nella directory enzo
sopracitata e lanciare il comando "python manage.py runserver"
A questo punto il portale è raggiungibile via browser all'indirizzo
http://127.0.0.1:8000

--------------------------------------------------------------------
Dipendenze

Python 2.7 e Django 1.8 devono essere installati sul sistema
Per installarli far riferimento ai rispettivi siti web
